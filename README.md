# Kotaro

心理学（Big Five）と神経科学を起点に、教育・医療領域のAIプロダクトを開発しています。

現在は、受験生向けAI学習SaaS「LEXUS Online」や、教材PDFから問題・解説・図版・動画を生成するLLM基盤を中心に、React / TypeScript、FastAPI、Supabase、Vertex AI / Gemini を用いたフルスタック開発に取り組んでいます。

診断、学習体験、生成AI、業務自動化をつなぎ、アイデアを実用可能なシステムまで落とし込むことが得意です。

## Focus

- AI学習SaaS / EdTech プロダクト開発
- 心理学・神経科学を用いた診断ツール開発
- 教材PDFからの問題生成・解説生成・図版生成
- LLM / マルチモーダルAIを使った業務自動化
- React / TypeScript / FastAPI / Supabase によるフルスタック開発

## Projects

### LEXUS Online

受験生向けのAI学習プラットフォームです。

問題演習、英単語・熟語、Elo難易度調整、SRS復習、チャット学習、決済、管理機能などを統合し、学習体験と運用基盤の両方を設計・開発しています。

### OCEAN-5

Big Five と神経科学をベースにした診断ツールです。

性格特性を単なるラベルではなく、学習・意思決定・行動設計に活用できる形へ落とし込むことを目指しています。

### AI教材生成基盤

教材PDFや画像を入力に、LLMと画像処理を組み合わせて、問題・解説・図版・動画を生成する仕組みを開発しています。

Manim、Penrose、RDKit、TikZ、PyMuPDF などを使い、数学・化学・物理・生物の教材生成に取り組んでいます。
